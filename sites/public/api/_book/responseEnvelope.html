响应包体说明

响应包体格式：

{
    "mid": "0",
    "cpmid": "1000",
    "result": "0"
  }

参数详细说明：

参数 参数说明
mid 下行流水号(备用ID，为0)
cpmid 商户接口提交时生成的cpmid,用于提交返回的匹配
result 返回结果，0代表成功，其他参数请参考返回值列表
